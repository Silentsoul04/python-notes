特殊属性

- instance.\_\_class\_\_

  类实例所属的类，可理解为当前实例的模板

  ```python
  >>> type('whale') # 字符串是str类的实例
  <class 'str'>
  >>> type(str) # str类是type类的实例
  <class 'type'>
  >>> type(type) # type类是type类的实例
  <class 'type'>
  ```

- class.\_\_bases\_\_

  类对象的基类构成的元组，只包含直接基类，并不是继承链上的所有基类

  ```python
  >>> bool.__bases__
  (<class 'int'>,)
  >>> int.__bases__
  (<class 'object'>,)
  ```

```
# 查看父类元组
>>> object.__bases__
()
>>> type.__bases__
(<class 'object'>,)
>>> int.__bases__
(<class 'object'>,)

# 查看模板
>>> object.__class__
<class 'type'>
```

```python
>>> def fn(self, name='world'):
	print('Hello, %s.' % name)

	
>>> Hello = type('Hello', (object,), dict(hello=fn))
>>> Hello.__bases__
(<class 'object'>,) #object是父类
>>> Hello.__class__
<class 'type'> #type是模板
```
